express框架

express是node.js的一个服务器端框架

下载安装express：npm install express

配置，接收请求，响应。

开始

与原生代码不同的是

非侵入式req

原生：

res.write();

res.end();

express:

res.send();

保留了原生功能，添加了一些方法（send）

使用

express已经将整个需要的模块与文件全部创建好了，只需要添加自己所需要的依赖和模块了。

创建一个目录，然后cd到这个目录下express app就完成创建后端框架了。
